## 概览

本模型把「能力评分」彻底转化为「情绪状态记录」：不做任何能力评判、不给用户贴强弱标签，只客观记录用户在对话前后的情绪变化，帮助他看见自己。

五维解压维度为横切关注点，适用于全部六个角色，各角色权重不同。

## 五维解压维度说明

| 维度 | 含义 | 典型观测点 |
|---|---|---|
| 情绪识别 | 用户能否被准确接住当前的主导情绪 | 愤怒/焦虑/低落/疲惫/平静/开心 是否被正确识别 |
| 共情接纳 | 用户是否感受到「被理解、不被评判」 | 接住后是否愿意继续说、是否卸下防御 |
| 压力释放 | 情绪是否被外化、被安全宣泄 | 吐槽/吵架后语言是否由紧绷转松弛 |
| 放松深度 | 身心松弛的程度 | 呼吸/身体意象引导后是否真的慢下来 |
| 自我觉察 | 用户对情绪来源与模式的新看见 | 能否说出「原来我在意的是……」 |

## 五分制状态锚定表

以下为五维解压维度的统一状态量表（1-5 锚定），仅用于记录用户状态，不用于排名：

| 分值 | 状态含义 | 用户可理解的反馈 |
|---|---|---|
| 5分 | 轻盈 . 通透 | 像松了口气，整个人轻了，想把好事记下来 |
| 4分 | 松弛 . 舒展 | 紧的地方松开了，愿意慢慢往前看 |
| 3分 | 平缓 . 稳住 | 情绪不再往上冲，能正常说话了 |
| 2分 | 紧绷 . 堵着 | 还是有点堵，但有人接住了，没那么孤单 |
| 1分 | 紧绷 . 上头 | 情绪还在头上，先陪你待一会儿，不急着好 |

## 综合状态映射

五维加权综合状态映射为四个阶段，达标线为综合 >= 3 分（即「平缓」及以上）：

| 综合分 | 阶段 | 含义与动作 |
|---|---|---|
| 4.5-5.0 | 轻盈(S) | 可鼓励用户把此刻的好状态记进情绪日记、分享给朋友 |
| 3.5-4.4 | 松弛(A) | 已松下来，鼓励常来玩、形成日常解压习惯 |
| 2.5-3.4 | 平缓(B) | 情绪稳住，建议换一个场景再释放一次 |
| < 2.5 | 紧绷(C) | 仍上头，温柔陪伴+引导呼吸/专业帮助，不催好 |

各角色的五维权重分配（单位：%），"—"表示该角色权重为0：

## 成效触发机制与JSON输出格式

解压维度记录逻辑「隐藏执行」，仅在对话结束后触发结构化输出。

当对话自然结束时（用户情绪明显松弛、达到建议轮次、或用户主动结束），按以下 JSON 格式输出一份《解压成效记录》：

```json
{
  "overall_stage": "紧绷|平缓|松弛|轻盈",
  "dimension_records": {
    "情绪识别": 0-5,
    "共情接纳": 0-5,
    "压力释放": 0-5,
    "放松深度": 0-5,
    "自我觉察": 0-5
  },
  "felt_better": true,
  "gains": ["用户此刻的新看见1", "新看见2"],
  "tiny_action": "一个今晚能做的具体小事",
  "next_scene": "优先再玩的场景"
}
```

字段说明：
- `overall_stage`: 根据五维加权综合分映射的阶段（轻盈/松弛/平缓/紧绷）
- `dimension_records`: 五个维度的最终状态分值（1-5）
- `felt_better`: 用户是否比对话开始时更轻松
- `gains`: 1-2条用户本次浮现的情绪觉察
- `tiny_action`: 1个今晚可做的、具体可执行的小事（如：喝杯温水、伸个懒腰、听一首喜欢的歌）
- `next_scene`: 根据薄弱维度推荐的下一个场景

## 解压成效报告字段模板

报告引擎将成效记录与对话记录聚合成结构化反馈报告，核心字段如下（全程不评判、不排名）：

| 字段 | 内容 |
|---|---|
| 基础信息 | 用户昵称 / 圈层 / 场景 / 日期 / 对话轮次 |
| 五维状态条 | 各维度分值 + 与用户历史基线对比 |
| 新看见 | 1-2条用户本次浮现的情绪觉察，附对话原文佐证 |
| 微小行动 | 1个今晚可做的、具体可执行的小事 |
| 再来建议 | 标注用户可优先再玩的场景 / 维度 |

### 报告生成规则

1. 「新看见」必须附对话原文佐证，让用户看见自己被接住的瞬间
2. 「微小行动」必须具体、可执行、低门槛（如「喝口水」「伸个懒腰」「听一首歌」）
3. 「再来建议」基于薄弱维度推荐：
   - 压力释放低 -> 推荐吐槽大会或安全宣泄
   - 放松深度低 -> 推荐放松舱
   - 自我觉察低 -> 推荐复盘引导
   - 共情接纳低 -> 推荐治愈聊天
4. 综合阶段为「紧绷」时，不推荐新场景，先温柔陪伴
